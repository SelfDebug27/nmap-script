### Requisiti

- Python 3.8 o superiore
- Nmap installato e presente nel PATH  
  (su Linux: sudo apt install nmap – su macOS: brew install nmap – su Windows: scaricalo da nmap.org)

### Installazione e uso

git clone https://github.com/SelfDebug27/script-nmap.git
cd nella cartella dove è stato clonato lo script
chmod +x script-nmap.py
./script-nmap.py
